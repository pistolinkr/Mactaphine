## 🛠️ Build Instructions

### macOS Native App
```bash
# Swift app build
./build_dmg.sh

# Result: Mac 데이터 클리너.dmg
```

### Cross-platform Electron App
```bash
# Install dependencies
npm install

# Build for all platforms
npm run build-all

# Build for Mac only
npm run build-mac

# Build for Windows only
npm run build-win
```
